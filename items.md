Revenue summary: MRR/ARR, new subscriptions, renewals, failed payments, refund count.

Content management status: About Us / Privacy / Terms last updated date and editor name.
User quality signals: flagged users/posts, spam reports, blocked accounts, moderation queue count.
Operational health: API uptime, error rate, failed jobs, DB latency, storage usage (uploads/media).
Geographic/device breakdown: users by country, platform split (iOS/Android/Web), app version adoption.
